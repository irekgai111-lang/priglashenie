# 📋 ПЛАН РАЗРАБОТКИ: TaxHelper РФ
## Умный ассистент для налоговых вычетов в России

**Версия:** 1.0  
**Дата:** 27 апреля 2026  
**Статус:** Готов к разработке  
**Целевой рынок:** Россия (самозанятые, фрилансеры, ИП, наёмные работники)  
**Платформа:** Telegram Bot

---

## 🎯 МИССИЯ И ВИДЕНИЕ

### Миссия
Помочь каждому россиянину получить максимальный налоговый вычет за 5 минут, используя шаблоны и формы ФНС РФ, без необходимости нанимать бухгалтера.

### Видение
Стать основным Telegram-ботом для подачи налоговых деклараций в России, интегрированным с реальным законодательством РФ и ФНС.

---

## 🇷🇺 КЛЮЧЕВЫЕ ТРЕБОВАНИЯ (РОССИЯ-СПЕЦИФИЧНЫЕ)

### 1. Соответствие ФНС РФ ✅
- ✅ Все расчеты по формулам из **Налогового кодекса РФ**
- ✅ Форма **3-НДФЛ** соответствует актуальной версии ФНС
- ✅ Реквизиты налоговых инспекций из **Единого реестра ФНС**
- ✅ Коды видов вычетов из **КБК (Классификатор бюджетных кодов)**
- ✅ Максимальные суммы вычетов обновляются каждый год

### 2. Отслеживание изменений закона 📰
- 📅 **Автоматический парсинг ФНС РФ** (portal nalog.gov.ru)
- 📰 **Мониторинг новостей налогового законодательства:**
  - Сайт ФНС (закон, приказы, письма)
  - КонсультантПлюс API (если доступен)
  - Telegram-каналы ФНС официальные
- 🔔 **Push-уведомления пользователям** при:
  - Изменении лимитов вычетов
  - Новых типах вычетов
  - Сроках подачи деклараций
  - Штрафах и льготах

### 3. Поддерживаемые вычеты РФ 💰

#### Социальные вычеты (Статья 219 НК РФ):
- **За лечение** (до 15,600₽ + 15,600₽ дорогостоящее)
  - Медицинские услуги в лицензированных учреждениях
  - Санаторно-курортное лечение
  - Страховые взносы на ДМС

- **За обучение** (до 13,000₽ для себя, 50,000₽ для детей)
  - ВУЗы, лицеи, школы с лицензией
  - Платные образовательные услуги

- **За взносы в пенсионные фонды** (до 120,000₽)
  - Добровольные взносы в ПФР
  - Взносы на НПО и УДО (УДП)

- **За спортивные услуги и фитнес** (до 15,600₽)
  - Физкультура и спорт
  - Санаторно-курортное лечение

- **За пожертвования** (до 25% дохода)
  - Благотворительные организации
  - Политические партии

#### Стандартные вычеты (Статьи 218 НК РФ):
- На детей (зависит от кол-ва и возраста)
- На инвалидов
- На боевых ветеранов

#### Имущественные вычеты (Статья 220 НК РФ):
- При продаже имущества
- При покупке/строительстве жилья (до 2,000,000₽)
- Проценты по ипотеке (до 3,000,000₽)

#### Профессиональные вычеты (Статьи 221-227 НК РФ):
- Для ИП и самозанятых
- Для авторов и изобретателей
- Для авторов ПО

#### Инвестиционные вычеты (Статья 224.1 НК РФ):
- По ИИС (до 400,000₽)
- При продаже ценных бумаг

### 4. Интеграция с ФНС РФ 🔗
- **Личный кабинет налогоплательщика ФЛ** (подача 3-НДФЛ)
- **Портал Госуслуг** (для подачи документов)
- **МФЦ** (альтернативный способ подачи)
- **Справка из ПФР** (для пенсионных вычетов)

## 📱 АРХИТЕКТУРА ПРОЕКТА

### Tech Stack (Россия-оптимизированный)

**Backend:**
```
Python 3.11 (FastAPI)
├── Database: PostgreSQL (ACID для финансовых данных)
├── Cache: Redis (сессии, rate limiting)
├── Queue: Celery (отправка документов, парсинг новостей)
└── ORM: SQLAlchemy (с поддержкой шифрования PII)
```

**Frontend/Bot:**
```
Telegram Bot (python-telegram-bot)
├── Conversational flow (вопрос → ответ → результат)
├── Inline buttons (для быстрой навигации)
├── File upload (загрузка документов, чеков)
└── Web app (для сложных форм на HTTPS)
```

**Integrations:**
```
├── ФНС РФ:
│   ├── Парсинг сайта nalog.gov.ru (Selenium/BeautifulSoup)
│   ├── Мониторинг налогового законодательства
│   └── API Госуслуг (через OAuth)
├── Финансовые интеграции:
│   ├── Яндекс.Касса (для фрилансеров)
│   ├── Сбер API (выписки, операции)
│   └── Мегафон (биллинг)
├── OCR и документы:
│   ├── Google Cloud Vision (распознавание чеков)
│   ├── LibreOffice (генерация PDF)
│   └── PDFLib (подписание документов)
└── Уведомления:
    ├── Telegram (основной канал)
    ├── Email (резервный)
    └── SMS (критические)
```

**Infrastructure:**
```
Яндекс.Облако (YC) — местный сервис для РФ
├── Compute Cloud (виртуальные машины)
├── Managed PostgreSQL (базу данных)
├── Managed Redis (кеш)
├── Object Storage (документы)
├── CDN (быстрая доставка в РФ)
└── Load Balancer (масштабирование)
```

### База данных (Schema)

```sql
-- Пользователи
users:
  - id (PK)
  - telegram_id (unique)
  - phone
  - passport_number (зашифрована!)
  - inn (ИНН)
  - snils (СНИЛС)
  - created_at
  - updated_at

-- Налоговые данные
tax_returns:
  - id (PK)
  - user_id (FK)
  - year (год декларации)
  - tax_regime (УСН, НПД, НДФЛ, ИП)
  - income (доход)
  - status (draft, submitted, accepted, rejected)
  - pdf_file (зашифрован!)
  - created_at
  - submitted_at

-- Вычеты
tax_deductions:
  - id (PK)
  - return_id (FK)
  - deduction_type (обучение, лечение, взносы, etc)
  - amount
  - documents (чеки, договоры)
  - verified_at (когда проверили документы)

-- Отслеживание изменений закона
tax_law_changes:
  - id (PK)
  - change_date
  - law_type (Налоговый кодекс, письмо ФНС, приказ)
  - description
  - affected_deduction_types
  - users_notified_at
  - source_url

-- История версий форм
form_versions:
  - id (PK)
  - form_type (3-НДФЛ, Приложение, etc)
  - year
  - version_number
  - pdf_template
  - fields (JSON)
  - released_date
```

## 📰 СИСТЕМА ОТСЛЕЖИВАНИЯ ИЗМЕНЕНИЙ ЗАКОНА

### Как работает мониторинг

**1. Автоматический парсинг (ежедневно)**
```python
# Парсим источники каждый день в 08:00 МСК
tasks:
  - parse_fns_nalog_gov_ru() # Сайт ФНС
  - parse_cbr_ru() # ЦБ РФ (важные письма)
  - parse_rospapir_ru() # Реестр страховщиков
  - fetch_telegram_fns_channel() # Официальный канал ФНС
  - check_nalogoviy_calendar() # Важные даты
```

**2. Детектирование изменений (AI/regex)**
```python
# Ищем ключевые слова в новых текстах
keywords = [
  "вычет увеличен", "лимит изменился",
  "новые вычеты", "отмена вычета",
  "штраф", "льгота", "уже нельзя",
  "максимальный размер", "изменение",
  "приказ ФНС", "письмо ФНС"
]

if any(keyword in text for keyword in keywords):
  -> SAVE TO tax_law_changes table
  -> NOTIFY users affected
```

**3. Классификация (какие пользователи затронуты)**
```
Изменение: "Максимум вычета на обучение повышен с 50K до 60K"
→ Затронутые: пользователи с вычетом "Обучение"
→ Действие: Отправить push "Отличная новость! Вычет за обучение..."
```

**4. Актуализация деклараций**
```python
# Если изменился лимит вычета
if deduction_limit_changed:
  -> Пересчитать все черновики пользователей
  -> Показать "Ваша декларация пересчитана!"
  -> Push-уведомление о новой максимальной сумме
```

### Примеры отслеживаемых изменений

| Изменение | Частота | Уведомление |
|-----------|---------|------------|
| Лимиты вычетов | Ежегодно | "Новые лимиты на 2026!" |
| Налоговые ставки | Редко | "Налоговая ставка изменилась!" |
| Новые вычеты | Пару раз в год | "Добавлен новый вычет!" |
| Сроки подачи | Ежегодно | "Срок подачи 4 апреля!" |
| Штрафы/льготы | Нерегулярно | "Внимание! Изменились штрафы" |
| Документы для подачи | При изменении | "Обновлены формы 3-НДФЛ!" |
