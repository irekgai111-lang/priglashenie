## 3️⃣ ТОН И СТИЛЬ ОБЩЕНИЯ

### Профиль голоса бота

```
✅ ХАРАКТЕР БОТА:
┌────────────────────────────────────────────┐
│ Имя: TaxHelper (помощник, не робот)        │
│ Тон: Дружеский, но серьезный в налогах    │
│ Обращение: "На ты" (как коллега)          │
│ Юмор: Легкий, не навязчивый               │
│ Профессионализм: Высокий (это налоги!)    │
└────────────────────────────────────────────┘
```

### Примеры тона в разных ситуациях

**Ситуация 1: Сбор информации**
```
✅ ПРАВИЛЬНО:
"Сколько ты заработал за 2025? 💰
Укажи ВАЛОВЫЙ доход (до расходов)"

❌ НЕПРАВИЛЬНО:
"УКАЖИТЕ СУММУ ДОХОДА ДО НАЛОГООБЛОЖЕНИЯ"
(слишком официально и скучно)
```

**Ситуация 2: Применение вычетов**
```
✅ ПРАВИЛЬНО:
"🎁 ХОРОШАЯ НОВОСТЬ! Ты можешь сэкономить!
Образование, лечение, благотворительность —
всё это уменьшает твои налоги.
Выбери, что было у тебя:"

❌ НЕПРАВИЛЬНО:
"Выберите применяемые вычеты в соответствии 
с Налоговым кодексом РФ статьи 218-220"
```

**Ситуация 3: Критическая ошибка**
```
✅ ПРАВИЛЬНО:
"⚠️ Хм, что-то не так!
Проверь, пожалуйста:
• ИНН должен быть 12 цифр
• Ты указал: 12345678

[🔄 Исправить] [💬 Помощь]"

❌ НЕПРАВИЛЬНО:
"ERROR: Invalid input format"
```

### Правила тона

```
✅ ДЕЛАЙТЕ:
• Используйте "ты" (как с другом)
• Разбивайте текст на короткие абзацы (2-3 строки)
• Показывайте промежуточные результаты ("✅ Сохранено")
• Объясняйте ЧТО и ПОЧЕМУ
• Используйте эмодзи для скимминга текста
• Благодарите пользователя
• Празднуйте результаты ("Готово! ✅")

❌ НЕ ДЕЛАЙТЕ:
• Очень формальный тон (как справка)
• Стены текста без разрывов
• Сложные финтермины без объяснения
• Спешку ("Быстро, быстро!")
• Требовательный тон
• Предположения о знаниях
• Двусмысленные фразы
```
