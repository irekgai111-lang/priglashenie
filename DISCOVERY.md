## 🎯 Бизнес-Модель

### Предложение:
1. **Пробный урок** (FREE)
   - Длительность: 60 минут
   - Цель: диагностика вокальных данных, выяснить возможности, понять причины проблем
   - Процесс: клиент выбирает день/время → ищем слот → приглашаем

2. **Основные занятия** (PAID)
   - Формат: индивидуальные
   - Расписание: гибкое, подбирается самим клиентом
   - Модель оплаты: нет абонентов, платишь за урок
   - **Пакет:** 8 занятий со скидкой при покупке в день пробного

### Целевая Аудитория (PRIMARY):
- Микс всех возрастов
- Взрослые и дети
- Начинающие с нуля
- Люди, которые хотят:
  - Попробовать петь
  - Выяснить свои вокальные данные
  - Сделать диагностику голоса
  - Узнать, почему не получается хорошо петь

### Primary Metrics:
- **KPI #1:** Регистрация на пробный урок (через сайт и другие каналы)


## 💡 Ключевые Идеи для Дизайна

1. **Эмоциональный посыл:**
   - Для стеснящихся: "Твой голос важен" / "Ты не один"
   - Для мечтающих: "От мечты к сцене" / "Голос, который тебя ждет"
   - Для родителей: "Развитие уверенности" / "Находка, которая изменит вашего ребенка"

2. **Цветовая палитра:**
   - Теплые, уютные цвета (голос = уют, безопасность, доверие)
   - Возможно: золотой, синий, теплый белый, черный

3. **Типография:**
   - Четкая, современная
   - Нужно читаемо на мобильном

4. **Структура:**
   - Hero (трансформация + фото + CTA)
   - О преподавателе (фото, био, квалификация)
   - Результаты студентов (отзывы, примеры)
   - Как это работает (процесс)
   - FAQ
   - Контакты + Final CTA
