# priglashenie — HTML приглашение на событие

## 📋 Описание
Веб-приложение для создания красивых пригласительных билетов на различные события (юбилеи, свадьбы, корпоративные события).

Первый проект - приглашение на 55-летний юбилей родителей в Нижнекамске.

## 🎯 Задача
- Создавать красивые пригласительные билеты в HTML формате
- Печать на фотобумаге (формат A6 или A5)
- Кастомизация текста, дат, реквизитов события
- Профессиональный дизайн с палитрой цветов

## 🎨 Особенности
- **Отзывчивый дизайн** — адаптируется под разные размеры
- **Печать** — оптимизирован для печати на бумаге
- **Кастомизация** — легко менять текст, цвета, шрифты
- **Масштабируемость** — можно создавать разные шаблоны для разных событий

## 🛠️ Технологии
- HTML5
- CSS3 (с переменными и Grid)
- JavaScript (опционально для интерактивности)
- Google Fonts (Prata, Cormorant Garamond, PT Serif)

## 📁 Структура
```
priglashenie/
├── PROJEKT.md          # этот файл
├── index.html          # главная страница (выбор события)
├── invitation.html     # шаблон приглашения (55-летие)
├── css/
│   ├── style.css       # основные стили
│   └── print.css       # стили для печати
├── images/             # фотографии объектов
└── .claude/            # конфигурация для Claude
```
