## 🔷 ГЛАВНЫЙ ПРОЕКТ: usn-helper

### Описание
Интерактивный веб-помощник для подачи налоговой декларации по УСН 6% в 2025 году. 
Приложение автоматически рассчитывает налоги, страховые взносы и позволяет применять вычеты.

### 📋 Основной функционал
- ✅ **8-шаговый wizard** для сбора данных
- ✅ **Автоматический расчет** налога 6% на доход
- ✅ **Страховые взносы** (53,658 ₽ за 2025)
- ✅ **Система вычетов**: обучение, больничные, НИОКР, благотворительность
- ✅ **Квартальные доходы** (Q1-Q4)
- ✅ **Список документов** для подачи
- ✅ **Пошаговые инструкции** ЛК ФНС
- ✅ **Экспорт печатного бланка** в PDF/HTML
- ✅ **Автоматическая проверка** обновлений размеров ФНС

### 🛠️ Технологии
- **Frontend:** React 18, Next.js 15, TypeScript, Tailwind CSS
- **Backend:** Node.js
- **Инструменты:** Bash скрипты для мониторинга ФНС

## 🟢 ДОПОЛНИТЕЛЬНЫЙ ПРОЕКТ: Приглашение на юбилей

### 🎨 Основной функционал
