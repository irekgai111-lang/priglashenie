### НЕДЕЛЯ 3-4: Быстрый Telegram-бот (v1)

**Архитектура (максимально простая):**
```
python-telegram-bot (библиотека)
├── Conversational flow (встроены вопросы)
├── SQLite (не PostgreSQL! простая файловая БД)
├── Google Forms для PDF генерации (или weasyprint)
└── Размещение на Heroku (бесплатно первые 1K часов)
```

**Код за 1 неделю (500 строк Python):**
```python
# handlers.py
def ask_deduction_type(update, context):
    buttons = [
        [InlineKeyboardButton("📚 Обучение", callback_data="edu")],
        [InlineKeyboardButton("🏥 Лечение", callback_data="med")],
        [InlineKeyboardButton("💰 Взносы", callback_data="contrib")]
    ]
    update.message.reply_text("Выбери вычет:", 
        reply_markup=InlineKeyboardMarkup(buttons))

# Дальше просто сохраняем ответы в SQLite
# Генерируем PDF
# Отправляем пользователю
```

**Deliverables:**
- Рабочий Telegram-бот (@TaxHelperRuBot)
- 3-5 основных вычетов
- Генерация простого PDF

**Бюджет:** $0 (Heroku free tier)  
**Результат:** 500+ юзеров

```

├── Heroku или Railway (простое развёртывание)

└── requests (парсинг ФНС простой скрипт)
```

**Код: 2,000-3,000 строк Python**

**Развёртывание: 1 час** (git push heroku main)

## ⚠️ РЕАЛЬНЫЕ РИСКИ (и как избежать)

| Риск | Вероятность | Решение |
|------|-----------|---------|
| Юзеры думают что это финансовый совет | Высокая | Дисклеймер на каждом шаге |
| Ошибка в расчёте = судебный иск | Средняя | Проверка экспертом, тесты |
| ФНС не нравится автоматизация | Низкая | Просто генерируем форму, юзер подаёт сам |
| Конкуренция vs TurboTax | Низкая | TurboTax не в России, мы локальные |
| Масштабирование | Средняя | Pre-warming, load testing в апреле |
| Пиратство (люди делают форк) | Средняя | Open-source, но с лицензией (AGPLv3) |

## 🎯 МЕТРИКИ КОТОРЫЕ ВАЖНЫ (не все)

**Не смотреть на:**
- ❌ Code quality (60% хорошего кода лучше чем 90% за 6 месяцев)
- ❌ Архитектура (рефакторим потом)
- ❌ UI дизайн (функциональность > красота)
- ❌ 100% тестов (70% важных тестов достаточно)

**Смотреть ежедневно:**
- ✅ **DAU** (Daily Active Users)
- ✅ **Completion rate** (% юзеров которые завершили)
- ✅ **MRR** (Monthly Recurring Revenue)
- ✅ **Churn** (% ушедших за месяц)
- ✅ **NPS** (Net Promoter Score - спрашиваем "порекомендуешь ли друзьям?")
