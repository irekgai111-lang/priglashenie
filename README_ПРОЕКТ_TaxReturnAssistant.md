# 📱 TaxReturnAssistant - Telegram Ассистент для налоговых вычетов НДФЛ

## 🎯 О ПРОЕКТЕ

**TaxReturnAssistant** - это Telegram-бот для помощи физическим лицам в получении налоговых вычетов НДФЛ. 

**Сколько денег можно вернуть?**
- 💰 **38 400 ₽** - среднестатистический пользователь
- 💰 **260 000 ₽** - при покупке квартиры
- 💰 **390 000 ₽** - при ипотеке

## 🏆 КОНКУРЕНТНЫЕ ПРЕИМУЩЕСТВА

### vs НДФЛка.ru
```
✅ Telegram интерфейс (удобнее)
✅ Пошаговой помощник (проще)
✅ Мгновенный расчет (быстрее)
✅ Мобильная-первая (актуально)
```

### vs Госуслуги
```
✅ Понятный интерфейс
✅ Помощь в заполнении
✅ Хранение документов
✅ Напоминания о сроках
```

### vs ВТБ Налоговый вычет
```
✅ Не требует открытия счета
✅ Доступен всем
✅ Быстрее
✅ Дешевле
```
