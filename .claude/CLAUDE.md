# Приглашение на юбилей — контекст для Claude

## ⛔ КРИТИЧЕСКИЕ ПРАВИЛА

### 1️⃣ НИКОГДА не удалять папку `.git`
- `.git` содержит всю историю проекта и связь с GitHub
- Без `.git` проект теряет связь с облаком

### 2️⃣ НИКОГДА не удалять эти файлы без явного разрешения
- `CLAUDE.md` — инструкции для меня
- `PLAN.md` — текущий план работы
- `SLOD.md` или другие документация `.md` файлы
- `.gitignore` — список игнорируемых файлов
- `.env.example` — пример переменных окружения

### 3️⃣ АВТОСОХРАНЕНИЕ перед выключением
- Перед каждым выключением компьютера я автоматически:
  1. Проверяю ВСЕ проекты в `/c/Users/Dell/Documents/project/`
  2. Делаю `git commit` если есть изменения
  3. Делаю `git push` в GitHub
  4. Сохраняю все PLAN.md, SLOD.md и .md файлы

### 4️⃣ ВАЖНЫЕ ПАПКИ (не трогать)
- `.claude/` — настройки и инструкции
- `.git/` — история и связь с GitHub
- `docs/` — документация
- `plans/` — планы проектов

### 5️⃣ ОБНОВЛЯТЬ ДОКУМЕНТАЦИЮ ВСЕ ПРОЕКТОВ
- После значимых изменений кода обновляю документацию (`PLAN.md`, `README.md`, `SLOD.md`, `PROJEKT.md`)
- Документация должна отражать текущее состояние проекта
- Особенно при изменении архитектуры, зависимостей или функций
