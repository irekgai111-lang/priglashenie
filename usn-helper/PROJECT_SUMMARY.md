# 📋 Полная информация о проекте "Помощник по подаче декларации УСН 6%"

## ✅ Что было создано

Полноценное веб-приложение на **Next.js + React + TypeScript + Tailwind CSS** для помощи в подаче декларации по УСН 6% за налоговый год 2025.

### 🎯 Основные компоненты

#### 1. **Архитектура приложения**
```
usn-helper/
├── app/                          # Next.js приложение
│   ├── layout.tsx               # Основной layout
│   ├── page.tsx                 # Главная страница с логикой
│   └── globals.css              # Глобальные стили
├── components/                  # React компоненты
│   ├── StepIndicator.tsx        # Индикатор шагов (1-7)
│   └── steps/                   # 7 компонентов для каждого шага
│       ├── Step1BasicInfo.tsx      # Основная информация
│       ├── Step2QuarterlyIncome.tsx # Доходы по кварталам
│       ├── Step3Insurance.tsx      # Страховые взносы
│       ├── Step4Documents.tsx      # Документы
│       ├── Step5Declaration.tsx    # Предпросмотр декларации
│       ├── Step6Instructions.tsx   # Инструкция по подаче
│       └── Step7Export.tsx         # Экспорт результатов
├── lib/                         # Утилиты и расчеты
│   ├── calculations.ts          # Функции расчета налога
│   └── types.ts                 # TypeScript типы
├── package.json                 # Зависимости
├── tsconfig.json               # Конфиг TypeScript
├── tailwind.config.js          # Конфиг Tailwind
├── postcss.config.js           # Конфиг PostCSS
└── next.config.js              # Конфиг Next.js
```

#### 2. **Технологический стек**
- **Framework**: Next.js 14
- **UI Library**: React 18
- **Язык**: TypeScript
- **Стилизация**: Tailwind CSS
- **Сборка**: Turbopack (встроен в Next.js)

#### 3. **Функциональность**

✅ **Многошаговая форма (7 шагов)**
- Шаг 1: Сбор базовой информации (ИНН, ФИО, наличие сотрудников)
- Шаг 2: Ввод доходов по кварталам + автоматический расчет налога
- Шаг 3: Информация о страховых взносах
- Шаг 4: Список необходимых документов для подачи
- Шаг 5: Предпросмотр заполненной декларации (разделы 2.1.1 и 1.1)
- Шаг 6: Пошаговая инструкция по подаче через ЛК ФНС
- Шаг 7: Экспорт результатов в текстовый файл

✅ **Автоматические расчеты**
- Налог = Доход × 6%
- Расчет по кварталам (Q1, Q2, Q3, Q4)
- Фиксированные страховые взносы (29 354 ₽)
- 1% от дохода свыше 300 000 ₽
- Уменьшение налога (полное без сотрудников, 50% с сотрудниками)

✅ **Пошаговая инструкция для подачи**
- Как войти в ЛК ФНС
- Как создать новую декларацию
- Как заполнить форму
- Как подписать ЭЦП
- Как отправить

✅ **Экспорт результатов**
- Скачивание текстового файла с полной информацией
- Форматирование для удобства

✅ **Учет всех подводных камней**
- Новая форма ФНС от 26.11.2025
- Разные правила для ИП с сотрудниками и без
- 50% ограничение при наличии сотрудников
- 1% взнос при доходе свыше 300k
- Срок подачи: 27 апреля 2026

## 🚀 Как использовать

### 1. **Запуск приложения**

Приложение уже запущено на **http://localhost:3000**

Откройте в браузере:
```
http://localhost:3000
```

### 2. **Пошаговое заполнение**

**Шаг 1:** Введите ИНН, ФИО, укажите наличие сотрудников
**Шаг 2:** Введите доходы за каждый квартал - система сама рассчитает налог
**Шаг 3:** Проверьте размер страховых взносов
**Шаг 4:** Прочитайте список документов для подачи
**Шаг 5:** Проверьте предпросмотр декларации
**Шаг 6:** Получите инструкцию по подаче через ЛК ФНС
**Шаг 7:** Экспортируйте результаты в файл

### 3. **Подача декларации**

После использования приложения:
1. Экспортируйте результаты (шаг 7)
2. Соберите документы (шаг 4)
3. Перейдите на lk.nalog.gov.ru
4. Следуйте инструкциям (шаг 6)
5. Подайте декларацию

## 💡 Ключевые особенности

### Расчет налога
```
Налог = Доход × 6%
(Рассчитывается по кварталам)

Страховые взносы:
- Фиксированные: 29 354 ₽
- 1% от дохода свыше 300k

Уменьшение налога:
- Без сотрудников: на полный размер взносов
- С сотрудниками: максимум на 50% налога
```

### Интерфейс
- Современный, чистый дизайн
- Градиентный фон (фиолетовый → синий)
- Адаптивный для мобильных устройств
- Индикатор прогресса (7 шагов)
- Все на русском языке

### Безопасность
- Все расчеты выполняются в браузере
- Данные не отправляются на серверы
- Соответствие всем требованиям ФНС

## 🐛 Troubleshooting

### Приложение не грузится
- Проверьте, что http://localhost:3000 доступен
- Убедитесь, что процесс Node.js запущен
- Проверьте, что порт 3000 не занят другим процессом

### Стили не применяются
- Очистите кеш браузера (Ctrl+Shift+Delete)
- Перезагрузите страницу

### Ошибки в консоли
- Откройте DevTools (F12 → Console)
- Посмотрите сообщения об ошибках
- Проверьте, что все файлы скомпилировались
