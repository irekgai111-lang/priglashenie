# 📋 УСН 6% Помощник — Декларация за 2025 год

Интерактивное веб-приложение для расчета и подготовки налоговой декларации по УСН 6% (доходы) за 2025 год.

## ✨ Возможности

- **8-шаговый wizard** — полный процесс от сбора данных до экспорта
- **Автоматические расчеты** — налог 6%, страховые взносы, вычеты
- **Система вычетов** — 4 типа вычетов для уменьшения налога
- **Интеллектуальное уменьшение** — автоматический учет правил (с/без сотрудников)
- **Предпросмотр декларации** — видите итоговый бланк перед отправкой
- **Пошаговая инструкция** — как подать через ЛК ФНС
- **Экспорт результатов** — HTML, JSON, CSV форматы
- **Проверка обновлений ФНС** — автоматическая проверка размеров взносов

## 📋 8 Шагов приложения

1. **Основная информация** — ИНН, ФИО, наличие сотрудников
2. **Доходы по кварталам** — Q1-Q4 2025, автоматический расчет итога
3. **Страховые взносы** — актуальные размеры на 2025 (53,658 ₽)
4. **Система вычетов** ⭐ **НОВОЕ** — выбор вычетов (обучение, больничные, НИОКР, благотворительность)
5. **Необходимые документы** — полный список документов для подачи
6. **Предпросмотр декларации** — готовый бланк с вычетами
7. **Инструкция по подаче** — пошаговая подача через ЛК ФНС
8. **Экспорт результатов** — скачивание HTML, JSON, CSV

## 📊 Расчеты

### Налог УСН 6%
```
Налог = Доход × 0.06
```

### Страховые взносы (2025)
- **ПФР**: 44,172 ₽
- **ФОМС**: 9,486 ₽
- **Всего**: 53,658 ₽

### Уменьшение налога с вычетами
```
К уплате = max(Налог - Взносы - Вычеты, 0)

Если есть сотрудники:
  max_reduction = Доход × 0.5 (не более 50%)

Если нет сотрудников:
  max_reduction = Доход × 1.0 (полное вычитание)
```

## 💡 Система вычетов

### 📚 Обучение
- Обучение работников или ИП
- **Максимум**: 120,000 ₽/год на одного работника
- **Требуется**: договор с учебным учреждением

### 🏥 Больничные выплаты
- Первые 3 дня болезни за счет работодателя/ИП
- Уменьшает налог на УСН
- **Требуется**: листки нетрудоспособности

### 🔬 НИОКР
- Научные исследования и разработки
- **Максимум**: до 50% от затрат
- **Требуется**: контракты на НИОКР

### 🤝 Благотворительность
- Пожертвования благотворительным организациям
- **Максимум**: до 25% от полученного дохода
- **Требуется**: документы о пожертвованиях

## ⚠️ Важная информация

1. **Срок подачи**: 27 апреля 2026 года (DEADLINE!)
2. **При опоздании**: Штраф 5-30% от налога (минимум 1,000 ₽)
3. **Новая форма**: Обновленный бланк ФНС с 26.11.2025
4. **ОКТМО**: Зависит от города регистрации
5. **Квартальные авансы**: Если не оплачены, включаются в итоговый налог

## 📚 Пример расчета

**Входные данные:**
- Доход: 500,000 ₽ (Q1-Q4)
- Сотрудников: 2
- Вычеты: Обучение 50,000 ₽

**Расчет:**
1. Налог: 500,000 × 0.06 = 30,000 ₽
2. Максимум вычета (с сотрудниками): 500,000 × 0.5 = 250,000 ₽
3. Вычитаем взносы: 30,000 - 53,658 = -23,658 ₽
4. **К уплате: 0 ₽** (налог полностью перекрыт взносами) ✅
