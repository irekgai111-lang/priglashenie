# Помощник по подаче декларации УСН 6%

Интерактивное веб-приложение для расчета и подготовки декларации по УСН 6% (доходы) за 2025 год.

## 🚀 Возможности

- **Пошаговый интерфейс** - 7 этапов от сбора информации до экспорта результатов
- **Автоматические расчеты** - налог, страховые взносы, уменьшение налога
- **Учет подводных камней** - правила для ИП с сотрудниками и без
- **Предпросмотр декларации** - видите как будут заполнены разделы 2.1.1 и 1.1
- **Пошаговая инструкция** - как подать через ЛК ФНС
- **Экспорт результатов** - сохранение в текстовом формате

## 📋 Шаги приложения

1. **Основная информация** - ИНН, ФИО, наличие сотрудников
2. **Доходы по кварталам** - введение доходов и расчет налога
3. **Страховые взносы** - фиксированные взносы и 1% от дохода
4. **Необходимые документы** - список документов для подачи
5. **Предпросмотр декларации** - заполненные разделы 2.1.1 и 1.1
6. **Инструкция по подаче** - пошаговая подача через ЛК ФНС
7. **Экспорт результатов** - сохранение расчетов в файл

## 🛠️ Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка

```bash
cd usn-helper
npm install
```

### Запуск разработки

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

### Сборка для production

```bash
npm run build
npm run start
```

## 📊 Особенности расчета

### Налог УСН 6%
- Налог = Доход × 6%
- Рассчитывается по кварталам

### Страховые взносы
- **Фиксированные взносы**: 29 354 ₽ на 2025 год
- **1% от дохода**: При доходе свыше 300 000 ₽

### Уменьшение налога
- **Без сотрудников**: Полное вычитание взносов
- **С сотрудниками**: Максимум 50% от налога

## ⚠️ Подводные камни

1. **Срок подачи**: 27 апреля 2026 года
2. **При опоздании**: Штраф 5-30% от налога (минимум 1 000 ₽)
3. **Новая форма**: С 26.11.2025 используется обновленный бланк ФНС
4. **ОКТМО**: Нужно указать в разделе 1.1 (зависит от города)
5. **Квартальные авансы**: Если не оплатили авансы, сумма включается в итоговый налог
