# 🚀 Быстрый старт УСН 6% Помощника

## ✨ Приложение 100% готово к использованию!

Помощник по подаче декларации УСН 6% полностью функционален с новой системой вычетов.

---

## 🌐 Запуск приложения

### 1. Установка (один раз)
```bash
cd usn-helper
npm install
```

### 2. Запуск разработчика
```bash
npm run dev
```

**Откройте в браузере:** **http://localhost:3003**

### 3. Production запуск
```bash
npm run build
npm run start
```

---

## 📱 8 Шагов использования

### Шаг 1️⃣: Основная информация
- **ИНН** — налоговый номер (10 цифр)
- **ФИО** — полное имя
- **Сотрудники** — есть ли они (влияет на налоговый вычет!)
  - 0 сотрудников → налог можно уменьшить на **100%**
  - 1+ сотрудников → максимум на **50%**

### Шаг 2️⃣: Доходы по кварталам
- **Q1** (янв-март): _____
- **Q2** (апр-июнь): _____
- **Q3** (июл-сент): _____
- **Q4** (окт-дек): _____
- Итог рассчитается автоматически ✓

### Шаг 3️⃣: Страховые взносы
Текущие размеры за 2025:
- **ПФР**: 44,172 ₽
- **ФОМС**: 9,486 ₽
- **Всего**: 53,658 ₽

Взносы автоматически уменьшают налог.

### Шаг 4️⃣: Вычеты ⭐ НОВОЕ!
Выберите применяемые вычеты:

#### 📚 Обучение
- Обучение работников
- Введите сумму (макс 120,000 ₽/год)
- Требуется: договор с учебным учреждением

#### 🏥 Больничные выплаты
- Первые 3 дня болезни за счет ИП/работодателя
- Введите сумму
- Требуется: листки нетрудоспособности

#### 🔬 НИОКР (научные исследования)
- Научные исследования и разработки
- Введите сумму (макс 50% затрат)
- Требуется: контракты на НИОКР

#### 🤝 Благотворительность
- Пожертвования благотворительным организациям
- Введите сумму (макс 25% дохода)
- Требуется: документы о пожертвованиях

### Шаг 5️⃣: Необходимые документы
Список документов для подачи:
- ✓ Сама декларация (из приложения)
- ✓ Документы подтверждающие доход
- ✓ Документы для вычетов (договоры, квитанции)
- ✓ Паспорт и ИНН

### Шаг 6️⃣: Предпросмотр декларации
Видите готовый бланк с:
- Вашими данными
- Расчетом налога
- Применяемыми вычетами
- **Итоговой суммой к уплате**

### Шаг 7️⃣: Инструкция по подаче
Пошаговые инструкции как подать через ЛК ФНС:
1. Вход на https://fl.lk.nalog.ru/
2. Выбрать отчетность → УСН 6%
3. Загрузить HTML бланк
4. Подписать и отправить

### Шаг 8️⃣: Экспорт результатов
Скачайте:
- **📄 HTML бланк** — печатный формат (откройте браузером, нажмите Ctrl+P)
- **📊 JSON отчет** — структурированные данные
- **📈 CSV отчет** — таблица для Excel

## ⚡ Быстрые ответы

### ❓ Где найти ИНН?
- На паспорте ИП
- На сайте ФНС: https://fl.lk.nalog.ru/

### ❓ Что если налог отрицательный?
**Это хорошо!** Значит взносы больше налога.
- Вы не платите налог
- Минус не возвращается (в общем случае)

### ❓ Нужна ли печать?
**Нет!** Можно подать онлайн через ЛК ФНС.
- Печать нужна только при личной подаче
- Используйте Ctrl+P в браузере для печати

### ❓ Когда подать?
⚠️ **ВАЖНО**: До **27 апреля 2026**!
- При опоздании: Штраф 5-30% от налога
- Подайте в тот же день!

### ❓ Где подать?
1. **Онлайн** (рекомендуется): https://fl.lk.nalog.ru/
2. **Лично** в ИФНС вашего города
3. **По почте** (заказное письмо)

## 📞 При проблемах

| Проблема | Решение |
|----------|---------|
| Приложение не грузится | Проверьте http://localhost:3003 и порт 3003 свободен |
| Стили не работают | Очистите кеш (Ctrl+Shift+Delete) и перезагрузите F5 |
| Ошибки в консоли | Откройте F12 → Console и сообщите об ошибке |
| Данные не сохранились | Данные хранятся в браузере, очистка кеша их удаляет |
