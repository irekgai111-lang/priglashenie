# 🚀 Быстрый старт

## Приложение готово к использованию!

Помощник по подаче декларации УСН 6% успешно запущен.

### 🌐 Доступ к приложению

**URL**: http://localhost:3001

Откройте этот адрес в браузере.

### 📱 Как использовать

#### Шаг 1️⃣: Основная информация
- Введите ваш ИНН (12 цифр)
- Введите фамилию и имя
- Выберите: есть ли у вас сотрудники

#### Шаг 2️⃣: Доходы по кварталам
- Введите доход за каждый квартал 2025 года
- Система автоматически рассчитает налог (6%)
- Увидите налог по кварталам и итого за год

#### Шаг 3️⃣: Страховые взносы
- Проверьте размер фиксированных взносов (по умолчанию 29 354 ₽)
- Система покажет 1% от дохода свыше 300 000 ₽

#### Шаг 4️⃣: Документы
- Прочитайте список необходимых документов
- Подготовьте все квитанции и справки

#### Шаг 5️⃣: Предпросмотр
- Проверьте заполненные разделы декларации
- Убедитесь, что все суммы верны

#### Шаг 6️⃣: Инструкция по подаче
- Получите пошаговую инструкцию для ЛК ФНС
- Все этапы подробно описаны

#### Шаг 7️⃣: Экспорт
- Скачайте текстовый файл с результатами
- Сохраните для своих записей

### ⚙️ Технические детали

- **Технология**: Next.js 14 + React 18
- **Стилизация**: Tailwind CSS
- **Язык**: TypeScript
- **Хранилище**: Состояние в памяти браузера

### 📝 Примечания

1. **Расчеты выполняются в реальном времени** - при изменении любого значения пересчитываются все суммы
2. **Данные в браузере** - все ваши данные хранятся только в браузере и не отправляются на серверы
3. **Экспорт** - используйте кнопку "Скачать" на шаге 7 для сохранения результатов

### 🔧 Команды разработки

```bash
# Запуск в режиме разработки (уже запущено)
npm run dev

# Сборка для production
npm run build

# Запуск production версии
npm run start

# Линтер (проверка кода)
npm run lint
```

### 📞 При проблемах

1. **Приложение не грузится** - проверьте, что порт 3000 свободен
2. **Стили не работают** - очистите кеш браузера (Ctrl+Shift+Delete)
3. **Ошибки в консоли** - проверьте браузерную консоль (F12 → Console)

### 🎯 Главные преимущества этого помощника

✅ Автоматический расчет налога по УСН 6%
✅ Учет всех подводных камней
✅ Правильное заполнение разделов 2.1.1 и 1.1
✅ Пошаговая инструкция для ЛК ФНС
✅ Экспорт результатов в файл
✅ Поддержка как ИП с сотрудниками, так и без
✅ На русском языке
✅ Бесплатно и без регистрации

### 📋 Что дальше?

1. Используйте помощник для подготовки вашей декларации
2. Экспортируйте результаты на шаге 7
3. Соберите необходимые документы (см. шаг 4)
4. Подайте декларацию через ЛК ФНС или лично в ИФНС
