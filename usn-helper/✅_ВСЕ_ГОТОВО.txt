🚀 ЧТО СОЗДАНО:
═══════════════════════════════════════════════════════════════════════════════

Полноценное веб-приложение для подачи декларации УСН 6%:

✅ Многошаговая форма (7 шагов)
✅ Автоматический расчет налога
✅ Учет страховых взносов
✅ Правильное заполнение разделов декларации
✅ Инструкция по подаче через ЛК ФНС
✅ Экспорт результатов в файл
✅ Красивый интерфейс на русском языке

ТЕХНОЛОГИЯ:
- Next.js 14 + React 18 + TypeScript + Tailwind CSS
