## 3. ДИЗАЙН-КОНЦЕПЦИЯ

### Тон общения: БАЛАНС между официальным и дружеским

```
✅ ДЕЛАЙТЕ:
  • Используйте эмодзи (но не переусложняйте)
  • Делите текст на короткие абзацы
  • Подтверждайте действия ("✅ Сохранено!")
  • Показывайте промежуточные результаты
  • Объясняйте, почему требуется информация

❌ НЕ ДЕЛАЙТЕ:
  • Очень официозный тон (как справка)
  • Стену текста без разрывов
  • Непонятные финансовые термины без объяснения
  • Спешку ("Быстро сделаем" без результатов)
  • Навязчивые рекомендации
```

### Эмодзи и визуальные элементы

| Элемент | Эмодзи | Использование |
|---------|--------|----------------|
| Информация | 💡 | Советы, подсказки |
| Доход | 💰 | Суммы, финансы |
| Налог | 💸 | Налоговые суммы |
| Документы | 📄📋 | Файлы, бланки |
| Успех | ✅ | Подтверждение |
| Процесс | ⏳⏭️ | Переход между шагами |
| Внимание | ⚠️ | Важная информация |
| Расчет | 📊 | Цифры, графики |
