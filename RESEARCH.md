# Research: Landing Page для Вокальной Студии "Твой голос"

## Дата: 2026-04-30

### 🎯 Ключевые Выводы из Research

#### 1. Структура Эффективного Музыкального Лендинга

**Above-the-fold (герой-секция):**
- Один мощный образ или видео, который заполняет весь экран
- Работает как "billboard" - должен сразу дать понять, что это правильное место
- Основной заголовок + CTA кнопка

**Основной посыл:**
- Фокус на ТРАНСФОРМАЦИИ, а не на технических деталях
- Родители/люди не волнуют теория музыки - их волнует результат
- Нужно нарисовать картину будущего: "Как будет звучать ваш голос после обучения"

#### 2. Критические Элементы для Конверсии

**Headline формула для музыкального лендинга:**
- `[Сервис] для [Аудитория] + [Трансформация]`
- Пример: "Научитесь петь с нуля без сложной теории за 2 месяца"

**Структура контента (по важности):**
1. ✅ Hero с трансформацией
2. ✅ Социальное доказательство (отзывы, результаты студентов)
3. ✅ Видимость преподавателя (фото, квалификация, опыт)
4. ✅ Четкие преимущества/особенности
5. ✅ Множественные CTA кнопки в разных местах
6. ✅ Форма захвата контактов или прямая ссылка на запись

#### 3. Лучшие Практики для Лендингов Уроков Музыки

**Headlines, которые работают:**
- Фокусируются на результате, а не на процессе
- Используют истории ("От страха к уверенности")
- Создают срочность ("Место ограничено")
- Обращаются к эмоциям ("Мечтали петь? Вот как")

**Видео:**
- Встроенное видео повышает конверсию на 69-100%
- Тип: демо-урок, интервью с преподавателем, отзывы студентов
- Идеально: видео преподавателя с кратким описанием методики

**Социальное Доказательство:**
- Отзывы студентов (текст + видео / текст + фото)
- Результаты: "Научился петь за 3 месяца", "Выступил на концерте"
- Количество: минимум 3-5 мощных отзывов выше fold'а

**Мобильность:**
- 100% мобильный дизайн - это обязательно
- Мобильная оптимизация может дать +27% конверсии
- Все CTA должны быть 44+ пикселя в высоту для click

#### 4. Воронка Конверсии для Музыкальных Уроков

```
Visitor lands → Sees transformation promise → Reads social proof → 
Sees teacher credentials → Clicks CTA → Takes action (Trial class/Contact)
```

**Ключевой момент:** Люди ищут знак, что они в правильном месте. 
Нужно это показать в первые 2 секунды.

#### 5. CRO Insights (Conversion Rate Optimization)

- **Baseline CTA без видео:** 6.5%
- **CTA с встроенным видео:** 11% (+69%) или 13% (+100%)
- **С социальным доказательством:** +24-31% конверсии
- **Мобильная оптимизация:** +27%

**Практический вывод:** Если добавить видео + отзывы + мобиль-оптимизацию, можем ожидать:
- Базовый сценарий: 6.5%
- С видео: 11%
- С видео + отзывы: ~14-15%
- С видео + отзывы + мобиль: ~18%+

## Применение к "Твой голос"

### Наши Преимущества:
✅ Сильный преподаватель (30+ лет певец, 20+ лет тренер)
✅ Уникальная методика
✅ Есть отзывы студентов
✅ Видимый результат (люди поют, участвуют в концертах)
✅ Доступная цена точка входа (бесплатный пробный)

### Что Нужно Реализовать:
- 📸 Профессиональное фото Ирека
- 🎥 Видео преподавателя (2-3 минуты про методику)
- 💬 3-5 мощных отзывов студентов
- 📱 100% мобильный дизайн
- 🎯 Четкая трансформация в headline ("От стеснения к сцене" / "От 0 до концерта")
- 🔄 Множественные CTA (минимум 3-4 кнопки на странице)
